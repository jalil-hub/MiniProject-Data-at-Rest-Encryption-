# Data-at-Rest Encryption Mini-Project

## Description
This project implements a secure file encryption system
to protect data stored on disk using AES encryption.

## Language
Python 3

## Libraries
cryptography

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Encrypt a file:
   python src/encrypt_file.py data/sample.txt

3. Decrypt the file:
   python src/decrypt_file.py data/sample.txt.enc
